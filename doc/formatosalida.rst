=================
Formato de salida
=================

El programa tiene como objetivo realizar un conjunto de calculos estadísticos sobre un conjunto de variables.

Por tanto, en los resultados debe aparecer la siguiente información:
 -Calculos realizados
 -Variables de actuación
 -Resultados numéricos

La implementación
-----------------

La salida tendrá una ventana propia. En esta ventana, irán apareciendo los diferentes resultados. Cada resultado es una tabla con la siguiente organización:

=================================
      Titulo de la operación
---------------------------------
          prueba1 prueba2 prueba3
========= ======= ======= =======
Variable1       x       x       x 
Variable2       x       x       x
Variable3       x       x       x
========= ======= ======= =======

Para permitir la exportación a html, el objeto que lleve cada resultado ha de tener una función que exporte el contenido a html
