=================
Ventana principal
=================

La ventana principal tiene dos tabs:

Atajos de teclado
-----------------

- F1 <- Ayuda
- F2 <- Tab de casos(registros)
- F3 <- Tab de variables
- F4 <- Diálogo de operaciones
- F5 <- Ventana de salida
- F10 <- Dialogo de configuración
- F12 <- Acerca
